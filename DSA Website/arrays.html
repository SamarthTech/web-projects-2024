Understanding Arrays

What is an Array?

An array is a collection of items stored at contiguous memory locations. It is a data structure that can hold multiple values of the same type.

Types of Arrays

  • 1D Arrays: A single row of elements.
  • 2D Arrays: An array of arrays, often used to represent matrices.
  • Multidimensional Arrays: Arrays with more than two dimensions.

Basic Operations on Arrays

Insertion

Adds an element to the array.

array.push(value);

Deletion

Removes an element from the array.

array.splice(index, 1);

Traversal

Accesses each element in the array.

for (let i = 0; i < array.length; i++) { console.log(array[i]); }

Searching

Finds an element in the array.

let index = array.indexOf(value);

Sorting

Arranges the elements in a specific order.

array.sort();

Example of a 1D Array

let fruits = ['Apple', 'Banana', 'Cherry'];
console.log(fruits[0]); // Output: Apple

Example of a 2D Array

let matrix = [
[1, 2, 3],
[4, 5, 6],
[7, 8, 9]
];
console.log(matrix[1][2]); // Output: 6
